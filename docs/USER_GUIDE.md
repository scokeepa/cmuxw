# cmuxw 사용자 가이드

`cmuxw` 사용 방법, 주요 명령어, 단축키를 한 곳에서 빠르게 확인할 수 있는 문서입니다.

## 빠른 시작

1. 앱 실행 후 워크스페이스를 선택합니다.
2. `Ctrl+T`로 새 서피스를 만들고, `Ctrl+D` 또는 `Ctrl+Shift+D`로 페인을 분할합니다.
3. 좌측 하단의 브라우저/알림/테마 기능으로 작업 흐름을 보조합니다.

## 자주 쓰는 CLI 명령어

```powershell
cmux workspace list
cmux workspace create --name "My Project"
cmux surface create
cmux split right
cmux pane list
cmux browser open https://github.com/scokeepa/cmuxw
cmux status
```

## 주요 단축키

- `Ctrl+N`: 새 워크스페이스
- `Ctrl+T`: 새 서피스
- `Ctrl+W`: 현재 서피스 닫기
- `Ctrl+D`: 오른쪽 분할
- `Ctrl+Shift+D`: 아래 분할
- `Ctrl+Shift+B`: 현재 페인 오른쪽에 브라우저 페인 열기
- `Ctrl+Shift+P`: 명령 팔레트
- `Ctrl+Shift+A`: 에이전트 채팅 토글
- `Ctrl+B`: 사이드바 토글
- `Ctrl+I`: 알림 패널 토글

## 브라우저 기능

- 브라우저 버튼/메뉴/단축키로 브라우저 페인을 즉시 열 수 있습니다.
- 기본 주소는 설정 > 동작 > `Browser Default URL`에서 변경할 수 있습니다.
- 주소창에 URL을 직접 입력해 즉시 이동할 수 있습니다.
